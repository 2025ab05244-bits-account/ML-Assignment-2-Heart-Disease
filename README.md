# ML Assignment 2 - Heart Disease Classification

## Problem Statement
Predict whether a patient has heart disease using classification algorithms.

## Dataset Description
Heart Disease Dataset
Features: 13
Target: Binary (0 = No Disease, 1 = Disease)

## Models Implemented
1. Logistic Regression
2. Decision Tree
3. KNN
4. Naive Bayes
5. Random Forest
6. XGBoost

## Evaluation Metrics
Accuracy, AUC, Precision, Recall, F1 Score, MCC

## Project Structure
- app.py
- requirements.txt
- README.md
- model/ (separate file per model)
